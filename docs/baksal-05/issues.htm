#1 Zielfeld eines Spielzuges kann nicht korrigiert werden.
2022-11-08 USP
Das State Transition Diagramm gibt im State 3 vor, dass das Zielfeld korrigiert werden kann (Kante "other destination selected"). Das funktioniert nicht.
Ursache: Die Markierung der gültigen Zielfelder ( Class Name "valdiDestination") wird bei der Auswahl des Zielfeldes (State 2, destination selected) gelöscht. Dadurch werden diese Felder nicht mehr als gültiges Zielfeld akzeptiert.
Workaround: Man kann das ausgewählte Tier anklicken (State 3, "animal selected"), dann werden die Markierungen erneut gesetzt.
Lösungsvorschlag 1: Einführung eines neuen CSS Class Attributes "highlightDestination", das mit dem Attribute "validDestination" gesetzt und mit der Auswahl eines Zielfeldes gelöscht wird. Das Attribut steuert über CSS die optische Hervorhebung der möglichen Zielfelder.
Lösungsvorschlag 2: Die Option "other destination selected" in State 3 wird ersatzlos entfernt.
