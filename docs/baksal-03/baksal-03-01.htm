Baksal 03-01
Hintergrundbild Studie

In dieser Studie wird eine Tabelle (TABLE) mit einer einzelnen Zelle (TD) dargestellt und mit einem Hindergrundbild versehen. So sieht's aus:

L

Bemerkungen

Hier ist der HTML Code:

<table id="spielbrett">
    <tr>	
        <td>L</td>
    </tr>
</table>

Der HTML Code zeigt zwar den Zelleninhalt "L", aber enthält offensichtlich keinen Hinweis auf das Hintergrundbild. Das ist richtig, denn das Bild ist in diesem Fall nicht "Content" wie das grosse "L", sondern es dient der grafischen Gestaltung. Es wird daher dem "Design" zugeordnet und mit einer CSS-Regel eingebunden:

table#spielbrett td {
    …
    background-image : url(pattern-22.svg); 
    background-size : contain ;
    }

Werfen wir einen Blick auf den Selektor der Regel. Man liest ihn zweckmäßigerweise von hinten nach vorne. Der Selektor adressiert also HTML TD-Elemente.

Das vorausgehende Leerzeichen ist, man höre und staune, ein CSS Operator! Also absolut nicht bedeutungslos wie sonst überall in der Programmierung. Er bedeutet "ist ein direktes oder indirektes Kind des linken Operanden".

Schauen wir uns also den linken Operanden an: table#spielbrett. Hier finden wir wieder einen Operator, das Hash (#). Es bedeutet mit einem ID-Attribut. Es geht also um TABLE Elemente mit einem ID Attribut "spielbrett". Das gibt es tatsächlich. Siehe Listing 1.

Es wäre zwar eigentlich nicht nötig gewesen, den Selektor so komplex zu gestalten, denn auf dieser Seite gibt es tatsächlich nur ein einziges TD element. Ein einfacher Selektor TD hätte also vollkommen gereicht. Aber einfach kann ja jeder.

Zusammengefasst: Der Selektor adressiert TD-Elemente in TABLEs mit der ID "spielbrett".

Werfen wir nun einen Blick auf die Attribut-Wertzuweisungen in dieser Regel. Da haben wir zunächst die Eigenschaft background-image, der der Wert url(pattern-22.svg) zugewiesen wird. Es ist also ein Link auf das Hintergrundbild.

Die nächste Zeile weist der Eigenschaft background-size den Wert "contain" zu, was bedeutet, dass das Bild den Container ausfüllen soll.

Das wars auch schon. Wie man oben sieht, hat man einen Kreis im Zentrum und ein paar Linien hindurch.

Ein paar Worte zum Bildformat. Es werden alle gängigen Internet-Bildformate akzeptiert, etwa PNG, SVG, JPG. Der Vorteil von SVG ist, dass es ein Vektorformat ist und sich verlustfrei auf beliebige Grösse skalieren lässt. Ausserdem ist es als XML-Format mit jedem Text-Editor editierbar. Also perfekt für diese Zwecke.
