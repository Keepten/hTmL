Anweisungen

Schafsprung: Klick auf ein Schaf, dann klicke auf das gewünschte Zielfeld. Wenn der Zug okay ist, klicke noch mal auf das Zielfeld um den Sprung auszuführen.

Willst du den Sprung korrigieren, klicke erneut auf das Schaf. Dabei wird zunächst das Zielfeld annulliert, dann das ausgewählte Schaf deselektiert und du kannst ein neues Schaf auswählen.
