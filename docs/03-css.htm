CSS

Wenn HTML den Inhalt und seine Struktur definiert, definiert CSS das Design (Schriftarten, Farben, Formatierung, Abstände, Textausrichtung, automatischen Zeilenumbruch, Sichtbarkeit, Rahmen, Schatten, Transformationen, Bewegungen, Animationen, und vieles mehr. CSS steht für "Cascading Style Sheets".

CSS Code besteht aus einer Liste von "Regeln", den CSS Rules. Eine Rule definiert, welche HTML elemente betroffen ("formatiert") werden, und wie.

CSS Rules bindet man vorzugsweise mit einem HTML LINK-Element in eine HTML-Datei ein:

<link rel="stylesheet" href="path/to/the/file.css" />

Spezielle CSS-Rules, die nur in einer einzigen HTML-Datei verwendet werden, können auch in einem HTML STYLE element definiert werden. Etwa so:

<html><head>

<style>
h1, h2, h3, h4, h5 { color : cornflowerblue }
h4, h5 { font-style : italic }
</style>

</head><body>
…
</body></html>

CSS Formatierungsregeln

Eine CSS Regel (Rule) besteht aus einem Selektor und einer Zuweisungsliste:

selector { zuweisung ; zuweisung ; ... }

Der Selektor "filtert" die HTML Elemente aus, auf die die Zuweisungsliste angewandt werden soll.

Beispiel:

p { font-family : "Times New Roman" }
h1, h2, h3, h4, h5 { color : cornflowerblue ; font-family : Arial, Calibri, Sans-Serif }

Das "p" filtert alle Paragraph-Element im Dokument aus. Ihr font-family-Attribut wird auf "Times New Roman" gesetzt. Anders ausgedrückt, die Schriftart für Absätze wird auf Times New Roman festgelegt.

Enthält ein Attributwert Leerzeichen wie etwa "Times New Roman", muss er in (einfache oder doppelte) Anführungszeichen gesetzt werden.

In der zweiten Zeile werden mehrere Selektoren aufgelistet. Sie filtern die Überschriften H1 bis H5 aus. Für sie wird die Schriftfarbe blau und der Font "Arial" festgelegt, alternativ Calibri oder Sans-Serif, falls Arial auf dem System nicht verfügbar ist.

Mehrere Zuweisungsangaben werden durch Semikolon getrennt. Mehrere Wertangaben werden durch Komma oder Leerzeichen getrennt:

img { margin : 10px 20px 10px 20px }

Die obige CSS-Regel legt die Abstände um die IMG elemente fest, in der Reihenfolge oben, rechts, unten und links.

Einige Wertangaben benötigen eine Einheit. Im obigen Beispiel wird die Einheit "Pixel" verwendet. Die Einheit darf nicht durch Leerzeichen vom Wert getrennt werden.

Häufig benutzte Attribute

Jedes HTML-Element bringt von Haus aus eine Unmenge von Attributen mit, die sein Aussehen und Verhalten bestimmen. Es ist sinnvoll, mit einem kleinen Satz von Attributen zu beginnen, und regelmäßig die Dokumentation der HTML-Element nach interessanten neuen Attributen zu durchforsten und auszuprobieren. Inkrementelles Lernen, sozusagen.

Die folgende Liste ist grob unvollständig — mit Absicht. Sie soll durch dich selbst ständig erweitert werden, in genau diesem Dokument.

Border
Viele, wenn nicht alle, HTML-Elemente haben eine Umrandung (Border), deren Stärke (Breite), Farbe (Color) und Muster (Style) man setzen kann. Beispiele:
border: 1px solid silver
border: 3px darkorange dotted
border-left: 3px darkorange dashed
border-bottom: 1px double blue
Eine Breite von 0 oder 0px lässt den Rand verschwinden. Besser aber ist es, den border-style=none zu setzen.
Siehe auch: MDN border-style, border-width, border-color.
Margin
Margin ist der Abstand zu anderen HTML Elementen, oben, rechts, unten und links. Beispiele:
style="margin-right:10px"
style="margin-right:2cm"
Wichtig: Der Margin zwischen 2 benachbarten Elementen wird nicht addiert, sondern verschmolzen. Der Abstand wird durch den größeren Wert bestimmt.
Die Werte können in verschiedenen Einheiten angegeben werden. Üblich sind Pixel (px), Point (pt), Zentimeter (cm) und Inch (in). Auch relative Angaben in Prozent (%) können gemacht werden, Bezugsgröße ist wohl die Seitenbreite.
Siech auch: MDN Margin.
Padding
Padding ist der Abstand zwischen Rahmen und Content. Syntax ist wie bei Margin.
Siech auch: MDN Padding.
Color
Textfarbe. Wertangabe als Web Color Name ("darkorange"), RGB hexadezimal dreistellig (#CCC), RGB hexadezimal sechsstellig (#C0B088), oder dezimal als RGB(128,0,255).
Siehe auch: MDN Color
Background Color
Hintergrundfarbe. Werte wie bei Color.
Siehe auch: MDN Background Color
Font Family
Das font-family-Attribut legt die Schriftart eines HTML Elementes fest. Es können mehrere Schriftarten angegeben werden. Wird die erste Schriftart im System nicht gefunden, wird die nächste versucht. Und soweiter.
Schriftarten enthalten oft Leerzeichen. In diesem Fall muss man den Namen in einfache oder doppelte Hochkommata setzen.
Siehe auch: MDN Font Family
Font Size
Das font-size-Attribut legt die Schriftgröße für HTML Elemente fest. Es muss eine Maßeinheit angegeben werden. Oft verwendet werden Point (pt), Pixel (px), Inch (in) und Zentimeter (cm).
Siehe auch: MDN Font Size
